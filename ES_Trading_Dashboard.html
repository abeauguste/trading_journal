Integration Guide
① TradingView Alert Setup
1. Open TradingView and add your ES Futures chart
2. Create a Pine Script indicator with VWAP, ATR, TTM Squeeze logic
3. Set an alert on your indicator condition
4. In the alert dialog, choose Webhook URL
5. Paste your server webhook URL
6. Use the JSON template from the panel above
② Webhook Server Options
Option A — n8n (Free): Self-host n8n, create a webhook workflow, store data to JSON file

Option B — Make.com: Free tier webhook → Google Sheet → this dashboard fetches it

Option C — Simple Python: 5-line Flask server to receive and cache alert payloads
③ Flask Receiver (5 lines)
from flask import Flask,request app=Flask(__name__) data={} @app.route('/webhook',methods=['POST']) def hook(): global data data=request.json return 'ok' @app.route('/data') def get(): return data app.run(port=5000)
